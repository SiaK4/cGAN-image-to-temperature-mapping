# cGAN-image-to-temperature-mapping
This repo contains the codes for the "Conditional GAN-based Image-to-Temperature Mapping of Vapor-to-Liquid Phase Change using Infrared Thermography Data" research paper by Siavash Khodakarami, Pouya Kabirzadeh, Chi Wang, Tarandeep Singh Thukral, and Nenad Miljkovic.

**The codes and datasets will be published soon.**
