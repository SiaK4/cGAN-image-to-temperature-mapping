# cGAN-image-to-temperature-mapping
This repo contains the codes for the "Mapping vapor-to-liquid phase change temperature dynamics via infrared thermography and generative machine learning" research paper by Siavash Khodakarami, Pouya Kabirzadeh, Chi Wang, Tarandeep Singh Thukral, and Nenad Miljkovic.

**The codes and datasets will be published soon.**
