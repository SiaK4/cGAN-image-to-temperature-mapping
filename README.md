# cGAN-image-to-temperature-mapping
This repo contains the codes for the "Optical to infrared mapping of vapor-to-liquid phase change dynamics using generative machine learning" research paper by Siavash Khodakarami, Pouya Kabirzadeh, Chi Wang, Tarandeep Singh Thukral, and Nenad Miljkovic.

**The codes and datasets will be published soon.**
